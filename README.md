# HireMind AI – Smart Interview Preparation Platform

## Setup

### Backend
```bash
cd backend
npm install
# Make sure MongoDB is running locally
npm run dev
```

### Frontend
Open `frontend/index.html` directly in a browser, or serve it with:
```bash
npx serve frontend
```

The frontend connects to `http://localhost:3000/api` by default.
All features work in demo mode even without the backend running.
